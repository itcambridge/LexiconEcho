EXECUTIVE STATUS VARIABLES (src/pages/api/consult.ts)
------------------
Status Flags:
  title: string                // Executive's title (e.g. 'Chief Technology Officer')
  isActive: boolean           // Whether executive is currently analyzing (controls border animation)
  hasResponded: boolean       // Whether executive has completed their analysis
  isPending: boolean         // Whether executive is waiting to start analysis
  timestamp: string          // ISO timestamp of when executive completed analysis
  error?: boolean           // Optional flag for if executive encountered an error
  errorMessage?: string     // Optional error message if executive encountered an error

API Response Types (src/pages/api/consult.ts):
  type: 'status'             // Event type for executive status updates
  type: 'response'           // Event type for individual executive responses
  type: 'final'             // Event type for final synthesis completion
  type: 'error'             // Event type for error conditions

Consultation Tracking:
  executiveStatuses: Record<string, ExecutiveStatus>  // Tracks all executives' current states
  executivesToConsult: string[]                      // List of executives to be consulted (from CEO)
  consultationResults: Array<{                       // Array of completed consultations
    title: string,                                   // Executive's title
    response: any,                                   // Executive's response data
    timestamp: string                                // When response was received
  }>
